## ⚙️ Como usar este repositório

1. **Clone o projeto:**
   `git clone https://github.com/seu-usuario/portfolio.git`

2. **Instale as dependências:**
   `npm install`

3. **Execute localmente:**
   `npm run dev`

4. **Deploy:**
   O projeto está configurado para deploy automático via Vercel ao realizar push na branch `main`.
